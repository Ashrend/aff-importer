# Arcaea资产管理工具

适用于Android平台的Arcaea资产管理工具。


## 功能

- ZIP 谱面包解析，自动提取元数据并追加到 `songlist`
- 对现有`songlist`元数据的预览与编辑
- 对现有`packlist`元数据的预览与编辑
